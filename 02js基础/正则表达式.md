# 正则表达式

## 一、基础语法速查

### 字符类

| 写法 | 含义 | 记忆方式 |
|---|---|---|
| `.` | 任意字符（除换行） | 万能通配 |
| `\d` | 数字 0-9 | digit |
| `\D` | 非数字 | 大写 = 取反 |
| `\w` | 字母数字下划线 | word |
| `\W` | 非字母数字下划线 | 大写 = 取反 |
| `\s` | 空白（空格/换行/tab） | space |
| `\S` | 非空白 | 大写 = 取反 |
| `[abc]` | a 或 b 或 c | 集合 |
| `[^abc]` | 不是 a b c | `^` 在 `[]` 内 = 取反 |
| `[a-z]` | a 到 z | 范围 |

> **记忆规律：小写 = 匹配，大写 = 取反**

---

### 量词

| 写法 | 含义 | 记忆方式 |
|---|---|---|
| `*` | 0 次或多次 | 可以没有 |
| `+` | 1 次或多次 | 至少一次 |
| `?` | 0 次或 1 次 | 可选 |
| `{n}` | 恰好 n 次 | 精确 |
| `{n,}` | 至少 n 次 | 最少 |
| `{n,m}` | n 到 m 次 | 范围 |

---

### 位置锚点

| 写法 | 含义 |
|---|---|
| `^` | 字符串开头（在 `[]` 外） |
| `$` | 字符串结尾 |
| `\b` | 单词边界 |
| `(?=...)` | 正向预查：后面是... |
| `(?!...)` | 负向预查：后面不是... |

---

### 修饰符

| 修饰符 | 含义 |
|---|---|
| `g` | 全局匹配（找所有，不加只找第一个） |
| `i` | 忽略大小写 |
| `m` | 多行模式（`^` `$` 匹配每行首尾） |

---

## 二、常用正则速查

```js
// 手机号
/^1[3-9]\d{9}$/

// 邮箱
/^[\w.-]+@[\w-]+\.[a-zA-Z]{2,}$/

// URL
/^https?:\/\/[\w-]+(\.[\w-]+)+([\w.,@?^=%&:/~+#-]*)?$/

// 身份证（简版）
/^\d{17}[\dX]$/

// 中文
/[\u4e00-\u9fa5]/

// 只含字母数字
/^[a-zA-Z0-9]+$/

// 密码（8-20位，含大小写和数字）
/^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)[a-zA-Z\d]{8,20}$/

// 去掉首尾空格
/^\s+|\s+$/g

// 连续空格替换为单个空格
/\s+/g

// HTML 标签
/<[^>]+>/g

// 提取数字
/\d+/g
```

## 三、JS 中的正则方法

```js
const str = 'hello world 123'
const reg = /\d+/g

// test：判断是否匹配，返回 boolean
reg.test(str)           // true

// match：返回匹配结果数组
str.match(/\d+/g)       // ['123']
str.match(/(\d+)/)      // ['123', '123', ...]  含捕获组

// replace：替换
str.replace(/\d+/, 'NUM')       // 'hello world NUM'（不加 g 只替换第一个）
str.replace(/\d+/g, 'NUM')      // 全部替换

// replace + 函数：动态替换
str.replace(/\d+/g, (match) => match * 2)  // 数字乘以2

// split：用正则分割
'a1b2c'.split(/\d/)     // ['a', 'b', 'c']

// search：返回第一个匹配的下标
str.search(/\d+/)       // 12

// matchAll：返回所有匹配（含捕获组），需要 g 标志
[...str.matchAll(/\d+/g)]
```

## 四、捕获组

```js
// () 捕获组：提取匹配内容
const date = '2024-01-15'
const [, year, month, day] = date.match(/(\d{4})-(\d{2})-(\d{2})/)
// year='2024', month='01', day='15'

// 命名捕获组：(?<name>...)
const { groups } = date.match(/(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})/)
// groups: { year: '2024', month: '01', day: '15' }

// (?:...) 非捕获组：分组但不捕获
'color colour'.match(/colou?r/g)  // ['color', 'colour']
```

## 五、贪婪 vs 懒惰匹配

```js
const html = '<div>hello</div><div>world</div>'

// 贪婪（默认）：尽可能多匹配
html.match(/<div>.*<\/div>/)
// ['<div>hello</div><div>world</div>']  一口吃掉整个

// 懒惰（加 ?）：尽可能少匹配
html.match(/<div>.*?<\/div>/)
// ['<div>hello</div>']  刚好够用就停
```

> 量词后加 `?` 变为懒惰模式：`*?` `+?` `{n,m}?`

## 六、面试常考题

**题 1：写一个正则，匹配千分位数字（如 1,234,567）**
```js
// 使用正向预查：后面有 n 组"3位数字"
const reg = /\B(?=(\d{3})+$)/g
'1234567'.replace(reg, ',')  // '1,234,567'
```

**题 2：实现一个模板字符串替换**
```js
const template = '你好，{{name}}，你今年{{age}}岁'
const data = { name: '张三', age: 18 }

template.replace(/\{\{(\w+)\}\}/g, (match, key) => data[key] ?? match)
// '你好，张三，你今年18岁'
```

**题 3：提取 URL 中的参数**
```js
function getParams(url) {
  const params = {}
  url.replace(/[?&]([^=&]+)=([^&]*)/g, (_, key, val) => {
    params[key] = decodeURIComponent(val)
  })
  return params
}
getParams('https://example.com?name=张三&age=18')
// { name: '张三', age: '18' }
```

**题 4：驼峰转下划线**
```js
'helloWorldFoo'.replace(/([A-Z])/g, '_$1').toLowerCase()
// 'hello_world_foo'
```

**题 5：去掉字符串中的非字母数字（本次虾皮面试题）**
```js
str.replace(/[^a-zA-Z0-9\s]/g, '')
```
