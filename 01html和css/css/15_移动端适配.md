# 移动端适配

## 核心概念

### 物理像素 vs CSS 像素

- **物理像素**：屏幕实际像素点数量，固定不变。
- **CSS 像素**（逻辑像素）：我们写 CSS 时用的单位。
- **设备像素比（DPR）** = 物理像素 / CSS 像素，Retina 屏 DPR = 2 或 3。

### viewport

```html
<!-- 标准移动端 meta 配置 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

| 属性 | 说明 |
|------|------|
| `width=device-width` | 视口宽度等于设备宽度 |
| `initial-scale=1.0` | 初始缩放比例 1:1 |
| `maximum-scale=1.0` | 禁止用户缩放（可选） |
| `user-scalable=no` | 禁止用户缩放（可选） |

## 适配方案

### 方案一：rem（最经典）

`rem` = 相对于**根元素（html）** 的 `font-size`。

```css
/* 设计稿 750px，html font-size 设为 100px */
html { font-size: 100px; }

/* 设计稿上 375px 的宽度 → 3.75rem */
.box { width: 3.75rem; }
```

**动态设置根字体大小（JS）：**

```js
function setRem() {
  const designWidth = 750; // 设计稿宽度
  const htmlFontSize = document.documentElement.clientWidth / designWidth * 100;
  document.documentElement.style.fontSize = htmlFontSize + 'px';
}
setRem();
window.addEventListener('resize', setRem);
```

**或用媒体查询静态设置：**

```css
@media (max-width: 375px) { html { font-size: 50px; } }
@media (max-width: 414px) { html { font-size: 55.2px; } }
@media (max-width: 750px) { html { font-size: 100px; } }
```

---

### 方案二：vw / vh（现代方案，推荐）

`1vw` = 视口宽度的 1%，无需 JS。

```css
/* 设计稿 750px，元素宽 375px → 375/750*100 = 50vw */
.box { width: 50vw; }

/* 字体大小 32px → 32/750*100 = 4.267vw */
.title { font-size: 4.267vw; }
```

可配合 `postcss-px-to-viewport` 插件自动转换，直接写 px。

---

### 方案三：媒体查询（响应式布局）

```css
/* 移动端优先（推荐） */
.container { width: 100%; }

@media (min-width: 768px) {
  .container { width: 750px; }
}
@media (min-width: 992px) {
  .container { width: 970px; }
}
@media (min-width: 1200px) {
  .container { width: 1170px; }
}
```

常见断点（参考 Bootstrap）：

| 名称 | 断点 |
|------|------|
| xs（手机） | < 576px |
| sm（平板）| ≥ 576px |
| md（小屏桌面）| ≥ 768px |
| lg（桌面）| ≥ 992px |
| xl（大屏）| ≥ 1200px |

## 1px 问题

Retina 屏（DPR=2）上，CSS 的 `1px` 实际渲染为 2 个物理像素，视觉上偏粗。

**解决方案：**

```css
/* 方案一：伪元素 + transform 缩放（推荐） */
.border-bottom::after {
  content: '';
  position: absolute;
  left: 0;
  bottom: 0;
  width: 100%;
  height: 1px;
  background: #ccc;
  transform: scaleY(0.5);
  transform-origin: 0 100%;
}

/* 方案二：viewport meta 缩放（影响整页） */
/* <meta name="viewport" content="width=device-width, initial-scale=0.5"> */
```

## rem vs vw 对比

| | rem | vw |
|--|-----|-----|
| 需要 JS | ✅（动态设置根字体） | ❌ |
| 兼容性 | IE9+ | IE11+ |
| 精确度 | 依赖根字体取整 | 精确 |
| 推荐度 | 老项目常见 | 新项目推荐 |
