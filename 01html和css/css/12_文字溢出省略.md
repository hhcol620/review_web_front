# 文字溢出省略

## 单行省略

```css
.ellipsis {
  white-space: nowrap;      /* 禁止换行 */
  overflow: hidden;         /* 隐藏超出内容 */
  text-overflow: ellipsis;  /* 显示省略号 */
}
```

三条缺一不可。

---

## 多行省略（WebKit 方案）

```css
.ellipsis-multi {
  display: -webkit-box;
  -webkit-box-orient: vertical;
  -webkit-line-clamp: 3;  /* 显示的行数 */
  overflow: hidden;
}
```

> 兼容性：现代浏览器均支持，IE 不支持。实际开发中已可放心使用。

---

## 属性说明

### `white-space`

| 值 | 换行 | 空格/Tab | 说明 |
|----|------|---------|------|
| `normal`（默认） | 自动换行 | 合并 | 正常 |
| `nowrap` | 不换行 | 合并 | 强制单行 |
| `pre` | 按源码换行 | 保留 | 类似 `<pre>` |
| `pre-wrap` | 自动+源码 | 保留 | 保留空格且自动换行 |
| `pre-line` | 自动+源码 | 合并 | 保留换行符 |

### `text-overflow`

| 值 | 效果 |
|----|------|
| `clip`（默认） | 直接裁剪，无提示 |
| `ellipsis` | 显示 `...` |

## 常见问题

### 宽度限制

容器必须有明确的宽度（或 `max-width`），否则省略号不生效：

```css
.ellipsis {
  max-width: 200px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
```

### 表格单元格中使用

`td` 默认会撑开宽度，需要额外设置：

```css
table { table-layout: fixed; }
td {
  max-width: 0;  /* 配合 table-layout:fixed 才生效 */
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
```
