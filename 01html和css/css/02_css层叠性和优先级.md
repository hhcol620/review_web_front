# CSS 层叠性与优先级

## 层叠性

多个选择器选中同一标签并设置同一属性时，浏览器只会采用一个值（其余被覆盖），这个过程由**优先级**决定。

---

## 优先级判断步骤

### 第一步：是否直接选中

- **直接选中**：选择器明确指向该元素。
- **间接选中**（继承）：通过祖先元素继承到的属性。

> 直接选中 > 间接选中（就近原则：离元素越近的祖先，继承优先级越高）

### 第二步：相同类型选择器

都是直接选中 + 相同类型选择器 → **后声明的优先**。

### 第三步：不同类型选择器

按选择器类型的固定优先级：

```
id选择器 > 类选择器 > 标签选择器 > 通配符选择器 > 浏览器默认样式
```

---

## 权重计算（混合选择器）

当多个选择器组合使用时，通过计算各类型数量来比较：

| 比较顺序 | 选择器类型 | 说明 |
|----------|-----------|------|
| 1st | `id` 选择器数量 | 多者优先 |
| 2nd | `class` 选择器数量 | id 数相同时比较 |
| 3rd | 标签选择器数量 | 前两者都相同时比较 |
| 最终 | 声明顺序 | 三者都相同时，后声明优先 |

> **注意**：只有**直接选中**才需要计算权重；若同时存在直接选中和间接选中，直接选中必然优先。

---

## `!important`

强制将某个属性提升到最高优先级。

```css
p {
  color: red !important;  /* !important 写在属性值后、分号前 */
}
```

**注意点：**

- 只能用于**直接选中**，间接选中（继承）无效。
- 通配符选择器也属于直接选中。
- 只提升**被标记属性**的优先级，其他属性不受影响。
- 慎用，滥用会使样式难以维护。
